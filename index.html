How to Add Your Own Map Images

Follow these steps to add your own map images to the project:

  1. Fork the repository by clicking the "Fork" button on the top right of the repository page.
  2. Clone your forked repository to your local machine using the command: 
    git clone https://github.com/YOUR-USERNAME/discord-utilities.git
  3. Navigate to the cloned repository directory: 
    cd discord-utilities
  4. Add your map image in the appropriate directory. The image should be in PNG format and named appropriately, e.g., `nazev_mapy.png`.
  5. Edit the `map_images.json` file to include your new map image entry. Add a new object to the JSON array with the name of your map image. For example: 
    {
    "name": "nazev_mapy",
    "file": "nazev_mapy.png"
}
  6. Commit your changes: 
    git add .
    git commit -m "Added nazev_mapy.png and updated map_images.json"
  7. Push the changes to your forked repository: 
    git push origin main
  8. Create a pull request to the original repository. Go to your forked repository on GitHub and click the "New pull request" button.

Once your pull request is reviewed and merged, your map image will be added to the project.
